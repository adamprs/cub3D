# cub3D

## Description

cub3D is a graphical project inspired by Wolfenstein 3D, aiming to recreate a first-person view inside a maze using ray-casting.

The program renders a 3D perspective from a 2D map, allowing the player to move through the environment in real time. It handles textures, colors, and user input to simulate a basic FPS experience.

This project focuses on graphics programming, event handling, parsing, and the use of mathematical concepts to implement efficient ray-casting algorithms.
